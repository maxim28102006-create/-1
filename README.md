# Telegram-бот для школьного расписания

Простой и понятный Telegram-бот для предоставления школьного расписания по номеру школы и класса с возможностью хранения данных и автоматической генерации ответов.

## Возможности

- Получение расписания по номеру школы и классу
- Просмотр расписания на конкретный день или на всю неделю
- Хранение расписаний в локальной базе данных SQLite
- Удобный интерфейс с клавиатурой для выбора дня недели
- Автоматическая генерация ответов

## Установка и настройка

### 1. Клонирование репозитория

```bash
git clone <url-репозитория>
cd maks
```

### 2. Установка зависимостей

Убедитесь, что у вас установлен Python 3.8 или выше.

```bash
pip install -r requirements.txt
```

### 3. Создание бота в Telegram

1. Найдите [@BotFather](https://t.me/botfather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Сохраните полученный токен

### 4. Настройка конфигурации

Создайте файл `.env` на основе `.env.example`:

```bash
cp .env.example .env
```

Откройте `.env` и вставьте ваш токен бота:

```
TELEGRAM_BOT_TOKEN=ваш_токен_здесь
```

## Запуск бота

```bash
python bot.py
```

При первом запуске бот автоматически создаст базу данных и добавит примерные данные для тестирования.

## Использование

### Команды бота

- `/start` - Начало работы с ботом
- `/help` - Справка по использованию
- `/schedule` - Получить расписание
- `/cancel` - Отменить текущую операцию

### Процесс получения расписания

1. Отправьте команду `/schedule`
2. Введите номер школы (например, `1`)
3. Введите класс (например, `9А` или `10Б`)
4. Выберите день недели из предложенных вариантов или "Вся неделя"
5. Получите расписание

## База данных

Бот использует SQLite для хранения расписаний. База данных автоматически создается при первом запуске.

### Структура таблицы `schedules`

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Уникальный идентификатор |
| school_number | INTEGER | Номер школы |
| class_name | TEXT | Название класса |
| day_of_week | TEXT | День недели |
| lesson_number | INTEGER | Номер урока |
| subject | TEXT | Название предмета |
| teacher | TEXT | ФИО учителя |
| classroom | TEXT | Номер кабинета |

### Добавление своих данных

Вы можете добавить свои данные расписания двумя способами:

#### 1. Через код Python

Откройте файл `database.py` и запустите:

```python
from database import ScheduleDatabase

db = ScheduleDatabase()
db.add_lesson(
    school_number=1,
    class_name="9А",
    day_of_week="Понедельник",
    lesson_number=1,
    subject="Математика",
    teacher="Иванов И.И.",
    classroom="201"
)
```

#### 2. Модификация примерных данных

Измените метод `add_sample_data()` в файле `database.py`, добавив свои данные.

## Разработка

### Добавление новых функций

Для добавления новых команд:

1. Создайте асинхронную функцию-обработчик в `bot.py`
2. Зарегистрируйте обработчик в функции `main()`

Пример:

```python
async def my_command(update: Update, context: ContextTypes.DEFAULT_TYPE):
    await update.message.reply_text("Ответ бота")

# В функции main():
application.add_handler(CommandHandler('mycommand', my_command))
```
